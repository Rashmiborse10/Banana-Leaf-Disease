Flask based BananaCare is an AI-powered banana leaf disease detection system developed using Python and deep learning techniques. 
The main goal of the project is to help farmers or agricultural users quickly identify diseases in banana plants by uploading an image of a banana leaf. 
The system uses a Convolutional Neural Network (CNN) model trained on banana leaf images to automatically classify different diseases based on visual symptoms such as spots, discoloration, or damaged leaf patterns.

The project was developed using libraries like OpenCV for image processing, NumPy and Pandas for data handling, and CNN architecture for feature extraction and classification. 
Data augmentation techniques such as image rotation, flipping, zooming, and scaling were applied to improve model accuracy and reduce overfitting.
 After training, the model was integrated into a web application using Flask, where users can upload leaf images and receive instant predictions.

The web application not only predicts the disease but also displays detailed information including disease description, possible causes, symptoms, and prevention methods.
 The system was tested on real-world banana leaf images to ensure practical usability and robustness under different lighting and background conditions. 
This project demonstrates the application of AI and computer vision in smart agriculture for early disease detection and crop health monitoring.
